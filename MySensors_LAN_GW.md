* Arduino NANO
* WizNET W5500 Module

| HT7333-A | Arduino NANO | WizNET W5500 | NRF24L01+ | wire color |
| ----- | ----- | ----- | ----- | ----- |
| GND   | GND   | GND   | GND  | sw  |
| VIN   | +5V   |       |      |     |
| VOUT  |       | 3.3V  | V+   | rt  |
|       | 10    | SCS   |      | or  |
|       | 11    | MOSI  |      | bl  |
|       | 12    | MISO  |      | vio |
|       | 13    | SCLK  |      | gn  |
|       | A0    |       | SCK  | gn  |
|       | A1    |       | MOSI | bl  |
|       | A2    |       | MISO | vio |
